## 1. 반복문 (While)
`while()` 문은 `()` 내 false값 들어오거나 내부 `break` 실행 될때까지 루프.

### 비교연산 방식
```java
// 비교연산 방식
int i = 0;
while(i < 10) { // 0 ~ 9 반복
    System.out.printf("i:%d ", i);
    i++; // 증가
}
```
### 탈출 방식 (break)
```java
// 탈출 방식
int i = 0;
while(true) {
    if(i == 10) break; // 0 ~ 9 반복
    System.out.printf("i:%d ", i);
    i++;
}
```
### do while 방식
```java
// do-while방식
int i = 0;
do {
    System.out.printf("i:%d ", i);
    i++;
} while (i < 10); // 0 ~ 9 반복
```
### continue 예시
```java
int i = 0;
while (i++ < 10) {
    if(i % 2 != 0) continue; // 홀수 제외 (건너뛰기)
    System.out.printf("i:%d ", i);
}
```
### 연습 1 ~ 100 합산
```java
int sum = 0, min = 1, max = 100;
int i = min;
while (i <= max) {
    sum += i;
    i++;
}
System.out.printf("합산 : %d%n", sum);
```

## 2. 반복문 (for)
`while()` 과 다르게 `for()`은 한 문장으로 루프 정의.
* `for(루프변수; 조건식; 증감식) {}`
*  `for(;;)` == `while(true)`

### 기본 방식
```java
for(int i = 0; i < 10;  i++) { // 0 ~ 9
    System.out.printf("i:%d, ", i);
}
```
### true 탈출 방식
```java
int i = 0;
for(;;) {
    if( i == 10) break; // 0 ~ 9
    System.out.printf("i:%d, ", i);
    i++;
}
```
### continue 예시
```java
for(int i = 0; i < 10; i++) {
    if(i % 2 != 0) continue; // 홀수 제외 (건너뛰기)
    System.out.printf("i:%d ", i);
}
```
### 이중 반복문
```java
for(int i = 0; i < 5; i++) { // 0 ~ 4
    System.out.printf("i:%d = ", i);
    for(int j = 0; j < 5; j++) { // 0 ~ 4
        System.out.printf("j:%d, ", j);
    }
    System.out.println();
    // i:0 = j:0, j:1, j:2, j:3, j:4
    // i:1 = j:0, j:1, j:2, j:3, j:4
}
```
